## المشاريع

### 1. Plan AI — منصة ذكية لتوليد مخططات UML

**النوع:** مشروع تخرج + مشروع شخصي
**الحالة:** قيد التطوير
**المدة:** 6 أشهر
**GitHub:** [github.com/IssamMekni/plan_ai](https://github.com/IssamMekni/plan_ai)
**تقدير المشروع:** جيد جداً

**الوصف:**
منصة ويب متكاملة (Full-Stack) تهدف إلى تسهيل وتسريع عملية إنشاء مخططات هندسة البرمجيات (UML Diagrams). تعتمد المنصة على تقنيات الذكاء الاصطناعي ومعالجة اللغات الطبيعية (NLP) لفهم متطلبات المستخدم المكتوبة بلغة طبيعية وتحويلها ديناميكياً إلى كود PlantUML ثم إلى مخططات دقيقة واحترافية.

**أبرز الميزات:**
- توليد المخططات بالنص (Text-to-Diagram)
- واجهة محادثة ذكية (Conversational UI)
- ميزات التعاون (Collaboration) مع Version Control
- معمارية قابلة للتوسع (Microservices)
- لوحة إدارة مستقلة بـ AdminJS

**التقنيات:**
- **محرك الرسم:** PlantUML Server API
- **الذكاء الاصطناعي:** Gemini API (Google) + Ollama (Local LLMs)
- **الواجهة الأمامية:** Next.js, React.js, Tailwind CSS, TypeScript (98.5%)
- **لوحة الإدارة:** AdminJS
- **الواجهة الخلفية:** Next.js API Routes, Prisma ORM, PostgreSQL, MinIO API
- **البنية التحتية:** Docker & Docker Compose
- **المنهجية:** Agile / Scrum

**معمارية النظام:**
```
User Browser
  ├─→ WebApp Container (Next.js)
  └─→ AdminJS Panel

WebApp ──→ PostgreSQL (Prisma Queries)
WebApp ──→ MinIO API (Upload/Download Files)
WebApp ──→ PlantUML Server API (Generate UML Diagrams)
WebApp ──→ AI Model API (AI Processing)

AdminJS ──→ PostgreSQL (Prisma Queries)
```

**التدفق:** `نص طبيعي` → `AI Model API` → `كود PlantUML` → `PlantUML Server API` → `مخطط جاهز`

**أنواع المخططات المدعومة:** Class Diagram, Sequence Diagram, Use Case Diagram

### 3. Productivity Launcher — تطبيق Android للإنتاجية

**النوع:** مشروع شخصي / Startup
**الحالة:** قيد التطوير — MVP (Q2 2026)
**المدة:** 3+ أشهر

**الوصف:**
تطبيق Android Launcher بديل يحول الهاتف الذكي من أداة تشتيت إلى بيئة إنتاجية مركزة. يعيد تصميم تجربة الشاشة الرئيسية بنظام تصنيف ثلاثي للتطبيقات، وآليات حماية متدرجة (Friction System) فريدة من نوعها.

**الميزات الأساسية:**
1. **نظام التصنيف الثلاثي:** تطبيقات صفحة رئيسية / بحث / مقفلة
2. **نظام القفل المتدرج (4 مستويات):** تأكيد → انتظار 15 ثانية → كتابة سبب → تحديات
3. **نظام البروفايلات:** قوالب جاهزة (طالب، موظف، مستقل، تركيز، استراحة)
4. **ويدجت الإنتاجية:** بومودورو، مهام يومية، متعقّب عادات
5. **تحليلات الاستخدام (محلية 100%)**

**التقنيات:**
- **Core:** Kotlin 100%, Jetpack Compose, Clean Architecture + MVVM, Hilt
- **Database:** Room Database, DataStore, Encrypted Storage
- **Android:** WorkManager, AlarmManager, UsageStatsManager, Custom Launcher API
- **UI:** Material Design 3, RTL Support, Dark/Light Theme
- **Testing:** JUnit, Mockito, Espresso, Turbine, Detekt

**الإنجازات التقنية:**
- Clean Architecture صارمة — سهولة إضافة ميزات
- أداء محسّن: LRU Cache (↓40% ذاكرة) + Lazy loading (↑60% وقت البدء)
- Local-first: لا cloud dependency، لا تتبع خارجي
- RTL كامل للعربية
